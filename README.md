# Jogo de Xadrez Interativo com Pygame

Este repositório contém um simulador de xadrez bidimensional completo, desenvolvido em Python utilizando a biblioteca Pygame. O projeto une lógica matemática e mapeamento de matrizes com o desenvolvimento de uma interface gráfica interativa baseada em eventos.

## 🚀 Funcionalidades e Características
* **Renderização em tempo real:** Interface gráfica dinâmica atualizada a 60 FPS.
* **Validação de Regras:** Codificação individual das regras de movimentação, vetores de ataque e caminhos de colisão para peões, torres, cavalos, bispos, rainhas e reis.
* **Gerenciamento de Estado:** Controle estrito de turnos de jogadas, interface para visualização de peças capturadas e destaque visual para peças selecionadas ou reis em situação de xeque.
* **Ciclo Orientado a Eventos:** Captura de interações do usuário através do mouse para manipulação fluida do tabuleiro.

## 🛠️ Tecnologias Utilizadas
* Python 3
* Pygame (Manipulação gráfica e eventos)

## 📋 Como executar
1. Certifique-se de ter o Python instalado.
2. Instale o Pygame executando: `pip install pygame`
3. Execute o projeto: `python main.py` (ou o nome do arquivo principal)
